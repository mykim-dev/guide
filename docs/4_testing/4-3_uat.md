# UAT (User Acceptance Test)

## 1. 대상 및 목표
- 대상: 내부 사용자(개발/디자인/QA)
- 목표: 문서 탐색 용이성, 컴포넌트 가이드 활용성 검증

## 2. 시나리오
- UAT-001: 홈→`/app-guide`→`/component-guide`→ 임의 슬러그 진입, 예제 확인
- UAT-002: Foundations 토큰 확인 및 다크/라이트 전환
- UAT-003: 플레이그라운드에서 속성 조작

## 3. 참여자 및 역할
- 사용자 대표 2, QA 1
