# 운영 매뉴얼 (Operations Manual)

## 1. 시스템 구성
- Next.js 정적 산출물 → GitHub Pages(또는 CDN)
- `basePath:'/guide'` 기준 경로 설정

## 2. 운영 절차
- 빌드/배포 파이프라인 실행, 산출물 검증, 모니터링

## 3. 모니터링/알림
- 가용성(헬스체크), 링크체커, 클라이언트 에러 로그(선택: Sentry)

## 4. 보안
- 비밀정보 미포함, 공개 저장소 검토

## 5. SOP/Runbook
- 링크 깨짐 발생 시: 로그 수집 → 원인 파일 슬러그 확인 → 수정/재배포

## 6. DR/BCP
- 태그/릴리스 백업, 재배포 스크립트 보관
